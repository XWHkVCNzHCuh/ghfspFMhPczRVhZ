🔍 Что такое Google Dorks?
ℹ️ Суть:
Использование продвинутых операторов поиска для обнаружения уязвимостей, скрытых файлов и конфиденциальных данных (OSINT).
🛠 Базовые операторы
🌐 Домен:
site:example.com — поиск только по конкретному ресурсу.
🔗 URL:
inurl:admin — поиск ключевых слов в адресе страницы.
📄 Файлы:
filetype:pdf, filetype:env, filetype:log — поиск документов и конфигов.
📝 Текст:
intitle:"index of" — поиск открытых директорий на сервере.
intext:"password" — поиск по содержимому страниц.
🛡 Примеры для Пентестинга
📂 Директории:
site:target.com intitle:"index of /backup"
🔑 Утечки:
site:target.com filetype:sql "insert into" — дампы БД.
site:target.com "root" "password" filetype:env — ключи окружения.
🚪 Админки:
site:target.com inurl:login | inurl:admin
⚠️ Ошибки SQL:
site:target.com intext:"SQL syntax error"
